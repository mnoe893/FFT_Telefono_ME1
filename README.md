# FFT_Telefono_ME1
Señales obtenidas por osciloscopio para las 9 teclas de un telefono de linea y el analisis de dichas medidas por FFT.

- `codigo.ipynb`: Bloque de codigo general, con el analisis conjunto de todas las teclas. 
- `analisis_individual.ipynb`: Codigo que analiza una señal a la vez
- `medidas`: Diretorio con los archivos .CSV de cada medicion adquirida por osciloscopio. Se tomaron 5 medidas por tecla, con el orden 1,2,3,4,5,6,7,8,9,*,0,#.
